# NotePadMMO
NotePadMMO is an experimental multiuser online document. It allows multiple people to edit in real time, simmilar to a google doc, except everyone online is on one big, open document. The server is coded in node.js, only relying on the send, ws, and http libs. A demo of this can be found [here](https://notepadmmo.science-dave-penguin.com) (don't expect perfect uptime as it is a demo, not a permenant installation).
