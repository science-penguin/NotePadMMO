# NotePadMMO
NotePadMMO is an experimental multiuser online document. It allows multiple people to edit in real time, simmilar to a google doc, except everyone online is on one big, open document. The server is coded in node.js, only relying on the send, crypto, fs, ws, and http libs. To start a server, run: node server.js in a directory containing server.js, main.js, client-home.html, client-404.html, and icon.png, which can be found in this project.
