NotePadMMO

NotePadMMO is a online free multiuser document. Anyone can type and you'll see their edits in real time on your screen. Move with arrow keys.
Note:

  • this relies on raw key capture, so mobile support is limited unless you attach a keyboard (mobile keyboad doesn't pop up).
  • Due to the fact that this display an entire megabyte of text (thats before browser objects ballon the actuall ram usage), it may lag on low end hardware.
  • Im not responsible for any user made content.
  • Don't expect perfect uptime, this is a small scale project subject to my personal life.
  • If the server unexpectedly goes down (ex: power outage, somone trips over my power cable, ect.), up to 100 seconds of edits will be lost, but all before that will be perserved.
  • Dont be mad if I ever decide to take this down, its just a small personal project.
  • Your edits show up on your screen as the server disributes them, not as you type, so there will be a slight delay (you see your edits the same time others do).
  • This project is open source (MIT license) and the source code can be found at https://github.com/science-penguin/NotePadMMO.
